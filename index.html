Project Details

This project applies advanced Natural Language Processing (NLP) techniques to tackle the challenging task of classifying text into two distinct categories: conspiracy narratives and critical thinking. By leveraging a range of machine learning models, including Support Vector Machines (SVM), Random Forest, and mBERT, the project conducts a comprehensive analysis of their performance across datasets in both English and Spanish.

Key findings highlight that Random Forest delivers the best results for English datasets, while SVM emerges as the most robust for Spanish. These outcomes underscore the significance of tailored preprocessing and feature engineering to address language-specific challenges, as models generally perform better in English. Text representation methods such as TF-IDF and FastText were evaluated, revealing that FastText embeddings particularly enhance performance with Random Forest.

Validation was meticulously conducted using K-Fold and Stratified K-Fold cross-validation techniques, demonstrating the reliability of the models and their generalization capabilities. Despite mBERT's potential as a transformer-based model, results showed that it introduced complexity in certain cases, requiring additional fine-tuning to maximize its utility.

Ultimately, the study emphasizes the importance of hyperparameter tuning and preprocessing in optimizing model efficacy. While no model achieved perfect classification, the combination of SVM for Spanish and Random Forest for English datasets offers a promising path for future applications in text classification tasks involving nuanced narrative detection. This research illustrates the growing potential of AI to discern subtle differences in language and foster a deeper understanding of diverse perspectives.

"Smart Pricing: The Airbnb Rental Advisor" introduces a cutting-edge tool designed to assist property owners in setting competitive Airbnb rental prices across ten popular European cities, including Amsterdam, Paris, and Rome. Built with Streamlit, the application combines a user-friendly interface with a robust Random Forest regression model to provide tailored rental price predictions based on user input. The application development involved several key stages:

  • Data Preprocessing: Comprehensive cleaning of Airbnb dataset features such as property type, location, and amenities, including handling missing values, encoding categorical data, and normalizing numerical variables.
  • Model Training: A Random Forest regression model was selected for its superior accuracy over other models like linear and lasso regression. Feature importance analysis and metrics like Mean Squared Error (MSE) and R² were used to fine-tune performance.
  • App Implementation: Streamlit was chosen for its simplicity, enabling real-time visualizations and dynamic user interactions. The app allows users to select city and property features, view price predictions, and receive tailored advice to optimize their rental strategies.

Key Features of the Application:

  • Price Prediction: Users receive accurate rental price recommendations based on city and property attributes.
  • Market Insights: Interactive graphs and visualizations showcase price distributions, influential features, and city-specific trends.
  • Personalized Advice: The app highlights actionable recommendations to enhance listing appeal and maximize revenue.
  • User-Friendly Interface: Designed for all users, regardless of technical expertise, ensuring accessibility and ease of use.

This innovative application addresses common challenges faced by property owners, such as pricing variability and market analysis, by leveraging data-driven insights. Future improvements include expanding to additional cities and platforms, integrating advanced machine learning algorithms, and incorporating user feedback for enhanced functionality. "Smart Pricing: The Airbnb Rental Advisor" exemplifies the potential of AI-powered tools to revolutionize decision-making in the rental property market, offering users a competitive edge in optimizing their Airbnb listings.

Project Details

APP LINK: https://movie-recommendation-system-svtgpgmbmqdeykq8mpl3fj.streamlit.app/

The Movie Recommendation System is a comprehensive project developed as part of a BSc in Computing with AI/ML at TU Dublin, Tallaght Campus. This project showcases the ability to integrate machine learning, data engineering, and web application development into a cohesive, user-centric solution.

The system addresses the challenge of finding personalized movie suggestions by leveraging a content-based approach. It utilizes the MovieLens dataset as its foundation, which is further enriched with metadata from the OMDb API. This enrichment process adds essential attributes like ratings, genres, plot descriptions, and poster URLs, ensuring the data's quality and relevance. Rigorous preprocessing was performed to clean, normalize, and structure the dataset, providing a robust base for the recommendation engine.

The recommendation engine itself employs TF-IDF vectorization and cosine similarity to rank movies based on their relevance to user preferences. By dynamically calculating similarities, the system delivers precise and relevant suggestions. Users can refine these recommendations through an intuitive and interactive web application built with Streamlit. The application allows users to explore and select movies, apply filters by genre, year, or IMDb score, and view personalized recommendations complete with detailed metadata and visuals like movie posters.

Evaluation of the system demonstrated its effectiveness, with a high precision of 0.7 in the top recommendations, scalability for datasets with thousands of entries, and an efficient runtime. Challenges such as inconsistent external data, API rate limits, and diversity in recommendations were successfully addressed through robust preprocessing, error handling, and strategic averaging across user selections.

Future enhancements for the system include the integration of hybrid recommendation techniques combining collaborative and content-based filtering, the use of advanced NLP models like BERT for deeper semantic analysis, and features such as user authentication for personalized profiles. Additionally, expanding the dataset with richer attributes like director, actor, and streaming platform availability is planned to further improve the system's versatility.

This project is a clear demonstration of practical problem-solving in the domain of personalized recommendations, emphasizing machine learning, data handling, and user-centric design principles.

Project Details

"Depicting Randomness" explores the intricate dynamics of particle motion through the lens of anomalous diffusion, a phenomenon where particles deviate from classical Brownian motion due to environmental interactions. Building upon the framework of the AnDi Challenge, this project leverages synthetic data and machine learning models to uncover the fundamental principles behind particle behavior.

The study focuses on two primary tasks: segmentation/classification and regression. Segmentation techniques identify shifts in particle states, using innovative features like "absolute displacement" and a rule-based model for state classification, achieving high accuracy with a Jaccard index of 1 and minimal Root Mean Squared Error (RMSE). For regression, the project predicts diffusion coefficients (K) and anomalous exponents (α) with various models, including Random Forest, SVM, MLP, and XGBoost. Notably, Random Forest excels in estimating α, while SVM achieves strong overall performance for both parameters.

Key results demonstrate robust model predictions with high R² values, highlighting SVM as the top-performing model. The project also underscores the importance of balanced datasets and feature engineering, with transformed variables (absolute displacement) significantly enhancing accuracy.

This research sets a strong foundation for future work in particle dynamics by making methods, datasets, and insights publicly available. It aligns with sustainability goals, offering applications in medical research, environmental conservation, and scientific innovation. With its interdisciplinary approach, the project contributes to advancing machine learning methodologies and understanding complex diffusion phenomena in real-world systems.
